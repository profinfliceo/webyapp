Compartir web

Para facilitar el seguimiento del proyecto, por parte del profesorado, se recomienda que las páginas, sin personalizar, las debe de crear el mismo. 

Una vez creada cada una de ellas, y conociendo los grupos correspondientes de alumnado para cada página y sus direcciones de correo, procedemos a compartir la misma con cada uno de ellos.

El proceso que tendremos que seguir es el siguiente:

1.- En la barra superior, pulsaremos sobre el icono de "Compartir":

Y nos mostrará la siguiente pantalla:

En el recuadro donde aparece "Añadir personar y grupos" escribimos la dirección de email del alumnado asignado a esta página.

En "Acceso general", para que el alumnado pueda hacer las modificaciones que considere debe estar marcado de la siguiente forma:

Al pulsar "Hecho", el alumnado asignado recibirá un correo electrónico para que puedan acceder a la web y modificarla.
