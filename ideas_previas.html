Definición de una página web

Una página web es un documento al que se puede acceder desde cualquier navegador con un acceso a internet que puede incluir diversos elementos como textos, imágenes, vídeos, audios, etc., combinado según se crea oportuno.

Existen diversos tipos de páginas: web estáticas y dinámicas, tiendas online, blogs, buscadores, etc.

Herramientas para crear web

Existen diversas herramientas para crear una página web.

Muchas de ellas requieren una programación textual utilizando código tipo html.

Para aquellos que no conocen este lenguaje de programación, existen alternativas muy sencillas, que nos permite crear web como si de un documento de texto se tratara.

Nos ofrecen diferentes posibilidades tales como blogs o web predefinidas.

Podemos desarrollar la web de nuestro proyecto con Blogger, Wordpress, Wix...

En nuestro caso, vamos a desarrollar una web con Sites de Google.

 

Proceso

Para llevar a cabo la página web de nuestro proyecto tenemos que realizar, antes de nada, un trabajo de campo, que se compone de las siguientes fases:

  • Agrupamiento del alumnado.
  • Asignación de los apartados del proyecto.
  • Búsqueda de información sobre la localidad (utilizando navegadores, guías turísticas, preguntando a los empresarios, etc.)
  • Realización de fotografías de los lugares de interés, locales de hostelería y comercio, etc.
  • Establecer un libro de estilo para todas las páginas web (encabezado, colores, tipo de letra, tamaño, disposición en la página, etc.)
  • Redacción de los textos en español y, posteriormente, en inglés.
  • Realización de los audios en ambos idiomas. Se realizará con el smartphone y se guardará en formato mp3.
